Welcome to the Craftplayer Project.

Craftplayer is a free, non-commercial, open framework for iOS <-> MinecraftSMP communication.

We're working on making a simple and functional API that lets you hook into any packet without having to parse them all. You provide the IP address, an username and a password (not required for offline mode servers), Craftplayer connects to the server and notifies your delegate about incoming packets, pre-parsed.

Demo PoC application

We at the Craftplayer project are working on something bigger, a full client. However, Craftplayer provides a solid backend for all your Minecraft SMP projects for iOS. We made a demo application that hooks chat and disconnect messages and let you chat on any Minecraft server , with any nickname/password. With just a few lines of code, we set up a connection and hooked chat (0x03) packets. We then receive a message to our delegate class that informs the UI about the message. Strings are automatically converted by the backend.

Need more support?

We know, developing an application off some sourcecode is generally hard without documentation. However, Craftplayer's so simple it requires little to no documentation at all. You could still encounter issues. For bugs, raise an issue ticket on Github. For general talk and implementation help, join our dedicated IRC Support channel over at irc.nightcoast.net #CraftMe. We'd love to have a chat with users of our project :)
